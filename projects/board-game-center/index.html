Board-Game-Center

Category:  Database , Web App , Flask

An application which can recommend you what board-games to play according to your current situation. Besides, it can do some statistics analysis.

Features
  • Recommend you board-games based on your situation. There are three filters:
    1. number of people
    2. the time length you would like to play
    3. game category
  • Search for board-game name using keyword.
  • There are three kinds of statistics about board-games. One of them is plotted dynamically according to the current data in our database.
Programming Languages/Frameworks
  • Python & Flask(back-end framework)
  • HTML & CSS & JavaScript
