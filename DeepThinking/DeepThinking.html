DeepThinking Documentation

🔶 Introduction

This app helps you to record and organize your ideas. It can lead you to think more deeply.

🔶 Functions

Block

  • A block can contain cards.
  • You can add a block, edit the name of a block, and delete a block.
  • You can press Ctrl+Space to add a new block.
  • In a block, you can add cards. When you are adding a new card and are typing, you can press Ctrl+Enter to add.

Card

  • You can edit the content of a card.
  • You can drag and drop a card to other blocks.

Export

Export as a text file. You can recover the blocks by importing the text file.

Import

Upload a text file exported in the past to import blocks.
